Download the COCO 2017 dataset and store it here.

  http://images.cocodataset.org/zips/train2017.zip

  http://images.cocodataset.org/zips/val2017.zip

  http://images.cocodataset.org/annotations/annotations_trainval2017.zip

  training images in `coco/images/train2017/` , val images in `coco/images/val2017/`, training annotations in `coco/annotations/`. Set the location of the dataset in the config file.
